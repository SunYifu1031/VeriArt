### 7. 使用说明

#### 查看数据
```bash
# 查看统计信息
python3 scripts/validate_data.py

# 查看具体数据
cat kg/cleaned_kg.json | jq '.nodes[] | select(.type=="Artist") | .label' | head -20
```

#### 更新数据
```bash
# 生成新数据
python3 scripts/scrape_artists.py

# 清洗和合并
python3 scripts/clean_data.py

# 验证数据质量
python3 scripts/validate_data.py
```

#### 前端测试
1. 启动后端: `cd backend && uvicorn app.main:app --reload`
2. 启动前端: `cd frontend && npm run dev`
3. 访问: http://localhost:5173
4. 测试搜索、浏览、统计等功能
